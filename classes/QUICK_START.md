## ğŸ“š Super Quick Learning

### Learn Python in 5 Minutes:
```python
# Print text
print("Hello!")

# Save information
name = "Hendrix"
age = 7

# Show the information
print("My name is " + name)
print("I am " + str(age) + " years old")

# Make choices
if age < 10:
    print("You're a kid!")
else:
    print("You're older!")

# Do things multiple times
for i in range(3):
    print("Coding is fun!")
```

### Learn HTML in 5 Minutes:
```html
<!DOCTYPE html>
<html>
<head>
    <title>Hendrix's Page</title>
</head>
<body>
    <h1>Welcome to My Website!</h1>
    <p>I'm learning to code!</p>
    <button>Click Me!</button>
</body>
</html>
```

### Learn Lua in 5 Minutes:
```lua
-- Print text
print("Hello from Roblox!")

-- Save information
local playerName = "Hendrix"
local coins = 100

-- Show the information
print("Player: " .. playerName)
print("Coins: " .. coins)

-- Make choices
if coins > 50 then
    print("You have lots of coins!")
end

-- Do things multiple times
for i = 1, 3 do
    print("Round " .. i)
end
```
